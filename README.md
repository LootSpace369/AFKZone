### Feature
You Can Customize How Much Money They Get! And Customize Worlds And Coordinates, Get Money Every Minutes.
(Now support EconomyAPI & BedrockEconomy)

### Commands
/afkzone <ui|setworld|setposition>
/settopafk
( You can use /az commands. )
